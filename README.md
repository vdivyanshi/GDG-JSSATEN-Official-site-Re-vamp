# GDG Official Website Redesign – UX/UI Case Study



## 📌 Overview

This project is a UX/UI redesign of the Google Developer Groups (GDG) website aimed at improving user engagement, navigation, event discovery, and community participation.

Google Developer Groups (GDGs) are community-led groups where developers learn, connect, and grow through events, workshops, and collaboration. :contentReference[oaicite:0]{index=0}

---

## 🎯 Problem Statement

The existing platform provides access to events, chapters, and community resources, but new users may find it difficult to:

- Discover relevant events quickly
- Understand the GDG ecosystem
- Join local chapters
- Navigate between programs and resources
- Engage with the community effectively

The goal was to create a cleaner, more intuitive, and visually engaging experience.

---

## 👥 Target Users

### Primary Users
- College students
- Beginner developers
- GDG chapter members
- Event attendees

### Secondary Users
- Organizers
- Industry professionals
- Community leaders
